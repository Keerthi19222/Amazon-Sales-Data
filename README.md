# Amazon-Sales-Data
This project focuses on analyzing Amazon sales data to extract meaningful insights and support data-driven business decisions. The analysis includes data cleaning, transformation, exploratory data analysis (EDA), and visualization to understand sales performance, customer behavior, and product trends

## 🎯 Objectives
- Identify top-performing products and categories  
- Analyze sales trends over time  
- Find top cities and states by sales  
- Understand customer purchasing behavior  

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- SQL
- Excel
- Power BI

---

## 📂 Dataset
The dataset includes:
- Order ID, Order Date  
- Product Category, SKU, Size  
- Sales Amount, Quantity  
- City, State  

---

## 🔄 Project Steps
1. Data Cleaning  
2. Data Analysis (EDA)  
3. Visualization  
4. Insights Generation  

---

## 📊 Key Insights
- Top categories generate highest revenue  
- Certain cities contribute most sales  
- Sales increase in specific months  

---

## 🚀 How to Run
1. Download dataset  
2. Run Python script / SQL queries  
3. Open Power BI dashboard
